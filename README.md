This project is a frontend UI implementation created as part of the HEDAMO Frontend UI Assignment.
It demonstrates how producer-declared product information can be presented in a clear, structured, and professional way.

The focus of this project is on transparency, usability, and clean interface design, not on validation or certification.

Project Overview

The application includes:

A product listing page with search and status filtering

A product detail page showing disclosure summary and version history

Clear disclaimer messaging to highlight producer responsibility

A calm, institutional-style user interface

All data used in this project is mock data for demonstration purposes only.
